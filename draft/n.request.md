## 功能
- 判断登陆状态
- 携带常用请求头并支持扩展
- 支持新旧的请求方式，例如 post DELETE
- 超时
- 错误统一处理并控制是否中断
- 成功和失败回掉
- 提供统一格式化处理环节
- 错误统计
- 过滤无效数据
- cookie 携带处理
- 错误提示是否忽略可配置化，多个同时报错时的控制报错个数
- 多个消息同时请求时处理，例如是否有共享的全局变量进行判断
